About our project

Welcome to the data science dashboard!

The dashboard provides a visualization of our data that has been collected from Twitter and Reddit.

The data can be visualized over two options:

  1. Tweet count by date: The program takes in a date or a date range as an input and plots a graph depicting the number of tweets collected everyday within or beyond the date range

  2. Popular Hashtags: The program takes in a "text topic" as an input and retrieves the top five, most popular hashtags related to the topic of search
